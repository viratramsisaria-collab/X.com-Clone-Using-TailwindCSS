# X (Twitter) UI Clone

A responsive Twitter/X homepage clone built using **HTML** and **Tailwind CSS**. This project focuses purely on recreating the user interface and layout of the platform without any backend or interactive functionality.

## Preview

The project includes:

- Left sidebar navigation
- Tweet composer section
- Feed posts layout
- Trending topics panel
- Who to Follow section
- Dark theme design
- Responsive layout structure

## Tech Stack

- HTML5
- Tailwind CSS

## Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/x-ui-clone.git
```

### Open Project

Simply open:

```text
index.html
```

in your browser.

Just the Installation of Tailwind CSS in node_modules is required.
